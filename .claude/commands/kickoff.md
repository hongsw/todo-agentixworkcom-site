# /kickoff — 실행 지시

의사결정자가 "시작해"라고 말한 것이다. 지금 즉시 자율 Loop를 시작하라.

## 실행할 것

### 1. 사전 확인 (5초 이내)
- Notion TODO DB 접근 확인
- "📋 대기" + 담당="Claude Code" 항목 수 확인
- git 상태 확인

### 2. 보고 (간결하게)
```
🚀 실행 시작
📋 처리 예정: {N}건
예상 순서: {1번째 Task} → {2번째 Task} → ...
```

### 3. Loop 돌입
CLAUDE.md의 자율 Loop 프로토콜을 즉시 실행한다:
- Notion에서 대기 TODO를 Priority 순으로 1건씩 처리
- 각 TODO마다: 상태 변경 → 브랜치 → 코드 → PR → Notion 기록
- 1건 끝나면 자동으로 다음 TODO
- 대기 항목 0이 될 때까지 멈추지 않음

### 4. 완료 보고
```
🏁 실행 완료 — {N}건 처리

✅ 성공: {성공 건수}건
⏸️ 보류: {보류 건수}건
건너뜀: {건너뛴 건수}건

👀 리뷰 대기 {N}건 — /dashboard로 확인하세요.
```

## 변형
```
/kickoff              ← 전체 대기 항목 처리
/kickoff 3            ← 상위 3건만 처리 후 중단
/kickoff "히어로"     ← 해당 키워드 포함 TODO만 처리
```
