# /check-feedback — 고객 피드백 확인

Notion TODO DB에서 고객의 코멘트와 미확인 항목을 확인합니다.

## 실행 순서

1. **리뷰 대기 항목 확인**: Status가 "👀 리뷰중"이고 "고객 확인" 체크박스가 비어있는 항목을 조회합니다.

2. **각 항목의 코멘트 확인**: Notion 페이지의 코멘트를 읽어 고객이 남긴 피드백이 있는지 확인합니다.

3. **피드백 분류**:
   - ✅ **승인**: "확인", "좋아요", "👍" 등 → 고객 확인 체크박스 체크
   - 🔄 **수정 요청**: 구체적인 변경 요청 → 새 TODO 생성 또는 기존 TODO 재오픈
   - ❓ **질문**: 추가 설명 필요 → 팀원에게 알림

4. **결과 요약 출력**:
   ```
   📬 고객 피드백 확인 결과:
   
   ✅ 승인됨 (2건):
   - 히어로 섹션 디자인 → 고객 확인 완료
   - 리드 수집 폼 → 고객 확인 완료
   
   🔄 수정 요청 (1건):
   - 제품 소개 섹션 → "이미지를 더 크게 해주세요" → 새 TODO 생성됨
   
   ⏳ 아직 피드백 없음 (3건):
   - 반응형 디자인, SEO 설정, GA4 설치
   ```

## 주의사항

- 고객 코멘트는 원문 그대로 보존합니다
- 수정 요청은 반드시 실행 가능한 TODO로 변환합니다
- 애매한 피드백은 팀원에게 확인을 요청합니다
