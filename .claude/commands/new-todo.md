# /new-todo — 새 TODO 생성

고객 요청이나 새로운 작업을 Notion TODO DB에 등록합니다.

## 사용법

```
/new-todo 히어로 섹션 버튼 색상을 빨간색으로 변경
```

또는 상세 정보와 함께:

```
/new-todo
Task: 히어로 섹션 CTA 버튼 색상 변경
Priority: 🟠 높음
Phase: 디자인
담당: Claude Code
설명: 고객이 기존 파란색 버튼을 빨간색으로 변경 요청
```

## 실행 순서

1. **입력 파싱**: Task명, Priority, Phase, 담당, 설명을 파싱합니다.
   - Task명만 있으면 나머지는 자동 추정합니다.
   - Priority 기본값: 🟡 보통
   - Phase 기본값: 현재 프로젝트 단계에 맞게 추정
   - 담당 기본값: Claude Code

2. **Notion TODO DB에 생성**: 새 항목을 추가합니다.
   - Status: 📋 대기
   - 작업 요약: Task명을 고객 친화적으로 재작성

3. **결과 출력**: 생성된 TODO의 Notion 링크를 표시합니다.

## 고객 코멘트에서 TODO 생성 시

고객이 Notion에 코멘트를 남겼을 때 이 커맨드를 활용합니다:

1. 코멘트 내용을 읽고
2. 실행 가능한 TODO로 변환
3. 원본 코멘트를 작업 요약에 인용

예시:
- 고객 코멘트: "로고가 너무 작은 것 같아요"
- 생성되는 TODO:
  - Task: 헤더 로고 크기 조정
  - 작업 요약: "고객님 피드백에 따라 헤더의 로고 크기를 키웁니다."
  - Priority: 🟡 보통
  - Phase: 디자인
