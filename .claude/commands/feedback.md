# /feedback — 고객 피드백 확인

Notion TODO의 고객 코멘트를 스캔하고, 의사결정자에게 대응이 필요한 것만 보여준다.

## 실행할 것

### 1. Notion 스캔
- Status="👀 리뷰중"인 항목의 코멘트를 전부 확인
- "고객 확인" 체크박스가 비어있는 항목 확인

### 2. 자동 처리 (의사결정자에게 묻지 않고)
- "확인했습니다", "좋아요", "👍" 같은 승인성 코멘트 → 자동으로 고객 확인 체크 + Status → "✅ 완료"
- 자동 처리한 내용은 보고에 포함

### 3. 의사결정자에게 보고

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💬 고객 피드백 — {날짜}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 자동 처리 ({N}건):
• {Task명} — 고객 승인 확인 → 완료 처리됨

🔔 대응 필요 ({M}건):
1. [{Task명}]
   고객: "{코멘트 원문}"
   → 수정 요청으로 보임. /approve 1 "수정 진행" 또는 /reject 1 "불필요"

2. [{Task명}]
   고객: "{코멘트 원문}"
   → 질문. 답변이 필요함. → /decide 2

⏳ 아직 피드백 없음 ({K}건):
• {Task명} — {완료일}에 리뷰 요청함 ({N일 경과})
```

### 4. 규칙
- 고객 코멘트 원문은 반드시 보존하여 보여준다
- 단순 승인은 자동 처리하고, 판단이 필요한 것만 의사결정자에게 올린다
- 3일 이상 피드백 없는 항목은 "⏳" 표시하여 리마인드
