# /reprioritize — 우선순위 재배치

대기중인 TODO의 실행 순서를 의사결정자가 변경한다.

## 사용법
```
/reprioritize                    ← 현재 대기 목록을 보여주고 재배치 요청
/reprioritize "리드폼을 먼저"    ← 자연어로 지시
/reprioritize 3 → 긴급          ← 특정 항목의 Priority 변경
```

## 실행할 것

### 1. 현재 대기 목록 출력
```
📋 현재 실행 순서 (대기 {N}건):

 순위  Priority   Task
 ─── ────────── ─────────────────────
  1   🔴 긴급    {Task명}
  2   🟠 높음    {Task명}
  3   🟡 보통    {Task명}
  4   🟡 보통    {Task명}
  5   🟢 낮음    {Task명}

변경할 내용을 말씀해 주세요.
예: "3번을 긴급으로", "리드폼을 1순위로", "5번 삭제"
```

### 2. 의사결정자의 지시를 받으면
- Notion TODO DB에서 해당 항목의 Priority를 변경한다
- 변경된 순서를 다시 출력한다:
```
✅ 우선순위 변경 완료:
  {Task명}: 🟡 보통 → 🔴 긴급

📋 새 실행 순서:
  1. 🔴 {Task명}
  2. 🔴 {Task명}
  3. 🟠 {Task명}
  ...

🚀 /kickoff로 실행하시겠습니까?
```
