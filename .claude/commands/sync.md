# /sync — Notion 동기화

현재 진행중인 작업의 상태를 Notion에 반영한다.
수동으로 코드 작업을 한 경우나, 상태가 맞지 않을 때 사용.

## 실행할 것

1. 현재 Git 브랜치와 최근 커밋을 확인
2. 관련 Notion TODO를 찾아 매칭
3. Notion 상태를 실제 상태에 맞게 업데이트:
   - PR이 있으면 → "👀 리뷰중" + PR 링크
   - 코드만 있고 PR 없으면 → "🔄 진행중"
   - PR이 merge되었으면 → "✅ 완료"
4. 작업 요약을 고객 언어로 업데이트
5. 결과 보고: "{N}건 동기화 완료"
