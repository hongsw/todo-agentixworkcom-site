# /approve — 승인

의사결정자가 항목을 승인한다. 나머지는 전부 너(Claude Code)가 처리한다.

## 사용법
```
/approve {번호 또는 Task명}
/approve {번호} "추가 지시사항"
/approve all          ← 리뷰중인 전체 항목 일괄 승인
```

## 실행할 것

### 단건 승인
1. 해당 Notion TODO의 Status를 "✅ 완료"로 변경
2. "고객 확인" 체크박스를 체크
3. 완료일이 비어있으면 오늘로 설정
4. 추가 지시사항이 있으면 새 TODO를 자동 생성 (Status="📋 대기", 담당="Claude Code")
5. 출력: "✅ 승인 완료: {Task명}"

### 일괄 승인 (approve all)
1. Status="👀 리뷰중"인 전체 항목을 "✅ 완료"로 변경
2. 모든 항목의 "고객 확인" 체크
3. 출력: "✅ {N}건 일괄 승인 완료"

### 후속 처리
- PR이 있으면 develop으로 merge가 가능한 상태인지 확인
- 대기 TODO가 남아있으면: "📋 대기 {N}건 남아있습니다. /kickoff로 실행하시겠습니까?"
